# Veterinary Clinic Website - Dr. Ikram

A modern, high-performance website for a veterinary clinic, built with Next.js and Tailwind CSS.

## Features

-   **Modern UI/UX**: Premium design with smooth animations and responsive layout.
-   **Internationalization**: localized content (French/English).
-   **Performance**: Optimized for speed with lazy loading, next/font, and proper image sizing.
-   **Contact Integration**: Direct WhatsApp integration for appointments.

## Tech Stack

-   **Framework**: [Next.js](https://nextjs.org/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **Animations**: CSS Keyframes & Framer Motion
-   **Deployment**: Vercel Ready

## Getting Started

1.  Current directory:
    ```bash
    npm install
    npm run dev
    ```

2.  Build for production:
    ```bash
    npm run build
    ```

## Deployment

This project is optimized for deployment on [Vercel](https://vercel.com).
Simply upload your project and deploy.
